# DiscOpti

Discord optimization toolkit for Windows. One-click setup that debloats Discord, applies performance tweaks, and installs optional mods (Equicord, OpenASAR, DiscOpt kernel).

## Download

Grab the latest release zip from [Releases](https://github.com/BarcusEric/DiscOpti/releases), extract it, then right-click `Disc-Optimizer.ps1` and choose **Run with PowerShell**.

## Usage

| Command | What it does |
|---------|--------------|
| `Disc-Optimizer.ps1` | First-time full setup (log in when prompted) |
| `Disc-Optimizer.ps1 -Launch` | Start Discord only (daily use) |
| `Disc-Optimizer.ps1 -Quick` | Re-apply tweaks after a Discord update |

## Requirements

- Windows 10/11
- PowerShell 7.7+ (installed automatically on first run)
- Administrator privileges (requested automatically)
