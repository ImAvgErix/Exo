# Exo Cua GUI QA

Live GUI verification of Exo using **[Cua Driver](https://github.com/trycua/cua)** (`cua-driver` 0.8.x on Windows).

## Why

Static XAML + smoke greps miss contradictions users feel (e.g. Discord **Already optimized** next to **not installed**). Cua walks UIA and captures screenshots against the **install-path** app without stealing focus.

## Prerequisites

1. Install Cua Driver (already on this PC):

   ```powershell
   irm https://cua.ai/driver/install.ps1 | iex
   ```

2. Exo installed at `%LocalAppData%\Exo\app\Exo.exe` (Start Menu).

## Run

```powershell
$env:PATH = "$env:LOCALAPPDATA\Programs\Cua\cua-driver\bin;$env:PATH"
pwsh -File tools/Cua.Qa/Invoke-ExoCuaQa.ps1
```

## Outputs

Generated by the script (gitignored screenshots optional — keep REPORT when useful):

- `REPORT.md` — element labels + honesty warnings
- `<module>.png` — window captures
- `*-state.json` — UIA snapshots
